#### Running locally

Install dependencies, run the app.

```bash
yarn install
yarn start # this opens the app in the default browser
```

Run tests

```bash
yarn test
yarn test --watchAll --coverage # print coverage report
```

#### Deploy to github pages

```bash
yarn deploy
```
